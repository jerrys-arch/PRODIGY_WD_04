# Personal Portfolio

This is a personal portfolio built using HTML, CSS, and JavaScript to showcase my skills, projects, and contact information.

## Sections

- **Home**: Introduction and brief about myself.
- **About**: Details about my background, skills, and expertise.
- **Projects**: A showcase of my recent projects and accomplishments.
- **Contact Me**: A form to get in touch with me.

## Technologies Used

- HTML5
- CSS3
- JavaScript

## How to Run

1. Clone this repository to your local machine.
2. Open the `index.html` file in your browser.

Feel free to explore and reach out through the contact section!
